ON DECENTRALIZED GOVERNANCE

Cypherpunks write code. We wrote the interface to a new kind of democracy.

In a world where power tends to centralize, we built tools for the opposite: trustless coordination, transparent treasuries, and permissionless participation.

OpenGov wasn't just a protocol upgrade‚Äîit was a statement. That governance can be open, continuous, and owned by no one.

We facilitated $140 million in collective decisions. Not through boardrooms or banks, but through cryptographic consensus and community conviction.

The experiment continues. The infrastructure remains. The code is open.
